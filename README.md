# Landpage - Portfólio Fotográfico

Site pessoal de portfólio fotográfico, viajante, fotógrafo e aventureiro. Desenvolvido com HTML, CSS e Bootstrap 5.

## Seções

| Seção         | Descrição                                                                |
| ------------- | ------------------------------------------------------------------------ |
| **Header**    | Cabeçalho fixo com logótipo, nome do autor e navegação entre secções.    |
| **Carrossel** | Slideshow automático com 5 fotografias de destaque (Bootstrap Carousel). |
| **Sobre**     | Apresentação do autor com vídeo do YouTube e texto descritivo.           |
| **Portfólio** | Galeria de 6 fotografias em grelha responsiva (3 colunas).               |
| **Contacto**  | Formulário de contacto e fotografia de perfil.                           |
| **Rodapé**    | Links para redes sociais, localização, telefone e e-mail.                |

---

## Tecnologias Utilizadas

- **HTML5** — estrutura semântica da página
- **CSS3** — estilos personalizados em `styles.css`
- **Bootstrap 5.3** — grelha responsiva, carrossel e componentes de formulário
- **Google Fonts** — fonte [Rock Salt](https://fonts.google.com/specimen/Rock+Salt) para títulos e logótipo

---

## Funcionalidades

- Cabeçalho fixo com navegação por âncoras
- Carrossel automático com controlos de navegação
- Galeria com efeito de zoom ao passar o rato (`hover`)
- Ícones de redes sociais com animação ao passar o rato
- Formulário de contacto com validação HTML nativa
- Design totalmente responsivo

---

## Como Executar

1. Clonar ou descarregar o repositório.
2. Garantir que a pasta `img/` contém todas as imagens listadas acima.
3. Abrir o ficheiro `index.html` num browser moderno.

> Não é necessário nenhum servidor ou instalação de dependências — o projeto corre diretamente no browser.
